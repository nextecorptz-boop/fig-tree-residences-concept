# Fig Tree Residences — Higgsfield Shot List
### Phase 4: Master Build Specification — v1.0

This is the per-image production list, building directly on the file-level inventory in `ASSET_INVENTORY.md` and the section-level briefs in Design Language's `10_HIGGSFIELD_PREP.md`. Every unique existing file is listed individually (duplicates already identified and excluded per Discovery's checksum work), followed by the new commissioned shots Visual Direction's priority list requires. Each row carries: Filename, Scene, Website section, Emotion, Lighting, Camera movement, Lens, Duration, Transition, Color grade, Ambient sound, Priority. For still images not being converted to video, Camera movement/Duration describe the on-site presentation treatment (static vs. a minimal Ken Burns drift), not a reshoot instruction.

## 1. Priority Legend

**Replace** — do not use as a final web asset; commission new photography per Visual Direction. **Correct-and-retain** — usable once colour-corrected per the neutral-wall test in `09_VISUAL_DIRECTION.md`. **Reference-only** — historical/content source, never displayed on the live site.

Every "Correct-and-retain" asset above must pass the neutral-wall test from `09_VISUAL_DIRECTION.md` before entering `08_IMAGE_OPTIMIZATION_PLAN.md`'s pipeline; assets that fail correction (cast will not fully resolve without visible artefacting) should be reclassified to Replace rather than shipped as a compromise, consistent with Visual Direction's standard that a shot must earn its place, not merely be the best of a weak set.
